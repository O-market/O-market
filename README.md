## 프로젝트 실행 방법

> 실행 방법

```bash
curl -Ls https://install.tuist.io | bash 또는
curl -Ls https://install.tuist.io | zsh

make
```

> Tuist 추가 명령어 (필요시 사용)

```bash
tuist fetch # 외부 라이브러리 Fetch
tuist generate # Workspace 및 Project 생성
tuist edit # 설정
tuist test # 테스트
tuist build # 빌드
tuist clean 
```
